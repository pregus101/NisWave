NisWave is a local media player that is still a work in progress with features and fixes coming out pretty often:

If you just want to run the program you'll need to do so either through and editor or a console. Make sure you install the requirements from the ```requirements.txt``` file

to run it on windows make sure your in the niswave directory then in the top path bar type cmd this will open command prompt (or you can just cd to it through cmd) then run 

```python niswave.py```

On linux and mac open a terminal and cd to the the NisWave directory and run

```python3 NisWave.py```
